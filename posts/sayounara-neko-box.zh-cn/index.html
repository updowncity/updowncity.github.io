再見NekoBox

2023-02-23
日誌
698?

睡醒一觉起来发现NekoBox提问箱因为不可抗力因素关闭了。虽然邮箱里还能找到收到的提问，但我的回复全都已经无法找回了。想到我开这个博客的契机也是NekoBox：好奇心使然地点开页尾的链接，找到开发者的个人博客，读到他毕业后写的关于大学四年的碎碎念，于是萌发了(再次)搭建个人博客的想法。这次很顺利，没有迷路。我想要一个initialization不那么轻松的个人博客，因为初始化时遇到的困难可以阻止我冲动之下把它销毁掉（我注销过很多个WriteFreely博客…）。NekoBox是我的第一个评论系统。既然开发者的Twitter简介里提到「不谈政治」，我也就不再这个post里谈论言论审查之类的话题了。

{% blockquote 一只大茄瓜🍆 @wuhan005 https://twitter.com/wuhan005/status/1628696994226257921 %} 今天突然接到网信办的电话，大致是我开发运营的一个站下面有人发敏感内容。
工作人员态度真心很好，他们的难处我也理解。配合完工作后还是关掉了运营三年的网站，凄惨。
一个人收拾收拾出门喝酒了。😢 {% endblockquote %}

这条Twitter让我想到去年这个时候想过的：虽然我们很厌恶state apparatus，但遇到的具体的人往往很nice。反抗行政机器就意味着要伤害这些曾经温柔地对待过我们的人们。（或许这种优柔寡断对其他没有孤独到放弃单数第一人称代词的人来说是难以理解的…）

{% blockquote 明流 @myouryu1 https://twitter.com/myouryu1/status/1628698895349084161 %} 😭😭😭刚点开就发现站无了，nekobox 真的非常优秀诶，推荐给了很多朋友，这样的结局让人很不舍……开发这样一个良好的平台很费心力，看日志和主页也能感受在上面花了很多力气和感情，对于开发它的您只能表达痛惜…… {% endblockquote %}

我也真的很羡慕这样能够自然地使用语言的人（以一种不那么perverse的方式传达自己的感情……）
