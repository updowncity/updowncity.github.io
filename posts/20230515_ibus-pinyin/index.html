圖形界面Ubuntu安裝拼音輸入法

2023/05/15 · 676 words · 2 minutes to read
Categories: 備忘錄
Tags: 圖形界面 Ubuntu 
OS Name: Ubuntu 22.04.2 LTS
OS Type: 64-bit
GNOME Version: 42.5

即便安装时选择的是Minimal installation，系统自带的ibus框架就已经很完整，所以可以通过图形界面安装（中国大陆使用者所熟悉的）拼音输入法。

打开Settings，安装新语言

雙擊Chinese (simplified) （我不知道选繁体中文会不会改变输入法…我只会用(大陸常見的)拼音，台湾人用的那種拼音我看不懂…） （感觉双击这個操作真的很反人類…我怎么也没办法激活那個Apply按钮…Enter没用，Ctrl+Enter也没用……然后暴躁地一通乱点发现双击就可以了。）

然後等待下載

有一個137.9MB的字體包，所以下載會比較慢（…我不太想換源。）

語言包下載过程中無法通过apt-get下載其他东西

下載完成後打開Settings，在Input Sources一欄下點加號添加新的输入法：

Add an Input Source裏選擇Chinese，打開就可以看到Intelligent Pinyin與Pinyin。（如果打開沒有出現这些选项就重啓下電腦。） 然後你就可以愉快地輸入中文了。右擊語言欄也可以切換簡繁/全角半角什麼的。默認快捷鍵和Windows那邊一樣。

不建議安裝的時候直接选中文（記不得是哪個Linux發行版了：选語言的時候直接选中文的話，它可能會給你直接用中文命名常用文件夾"Documents",“Downloads”,“Music”,“Picturess"等文件夾。（與Windows不太一樣：Windows只是圖形界面里顯示中文，文件路徑依舊是ASCII。）
