你快樂嗎

2023-05-19

到目前为止的五月简直糟透了（我又有哪个月不是这样糟透了的呢，除了那梦幻般的几个例外…）毕业论文写到最后就已经对这个topic感到厌倦，只想早点结束然后做点别的。临近答辩，很焦虑但一点做事的动力也没有。一直拖到（……）。本以为这件事到此结束……。 网站坏了。经常出现「隐私错误无法访问」的故障。删掉CloudFlare，Vercel以及Netlify的仓库，到Namesilo的域名管理页删掉相应的DNS记录，重新部署GitHub pages。（……我真的不懂Web…所以只能这样解决问题。）

再然后，可怕的事情发生了。从9号上午，到13号中午：冲动是魔鬼。9号早上突发奇想开始搞双系统。忘记备份电脑里的照片与文件。用的是最新的Ubuntu 23.04（……小白为什么不选LTS），它无法识别电脑里的Windows系统，我忘记了自己还没备份这件事，于是放弃双系统直接安装……。（然后才意识到我的毕业论文终稿就这么消失了。）23.04似乎有点问题，安装TexLive以后就怎么也开不了机了。只好用U盘里的镜像从头再来。然后安装了TexLive发现它又开不了机了。又一次重来，下载新的Windows 10镜像。蜂窝数据不够用，只好充校园网。然而校园网下不动Windows镜像（从2022年开始，有时用校园网只能访问国内已备案/服务器在国内的网站…）。（TexLive与Ubuntu可以用清华的镜像站…）印象里之前是用WoeUSB做的，直接用dd似乎有点问题，因为超出了文件大小上限。然而这次WoeUSB总是安装失败。碰运气般地试dd，虽然能够进入Windows安装界面，但依旧失败，继续碰运气，找各种经验贴，终于找到一个能用的。 打完论文终稿已经是13号凌晨三点。然后睡过了头，险些错过回家的高铁。

后来意识到终稿丢了其实真的只是一件小事，两个通宵就可以搞定。但是我从2019到2021的照片彻底无法找回了：我对着密码管理器登录所有注册过的网盘，都没有找到。对数据恢复不报希望（……装过五遍系统。）

圖形界面Ubuntu安裝拼音輸入法

2023-05-15 備忘錄 
OS Name: Ubuntu 22.04.2 LTS
OS Type: 64-bit
GNOME Version: 42.5

即便安裝時選擇的是Minimal installation，系統自帶的ibus框架很完整，所以可以通過圖形界面安裝（中國大陸使用者所熟悉的）拼音輸入法。

打開Settings，安裝新語言

雙擊Chinese (simplified) （我不知道選繁體中文會不會改變輸入法…我只會用(大陸常見的)拼音，臺灣人用的那種拼音我看不懂…） （感覺雙擊這個操作真的很反人類…我怎麼也沒辦法激活那個Apply按鈕…Enter沒用，Ctrl+Enter也沒用……然後暴躁地一通亂點發現雙擊就可以了。）

然後等待下載

有一個137.9MB的字體包，所以下載會比較慢（…我不太想換源。）

語言包下載過程中無法通過apt-get下載其他東西

下載完成後打開Settings，在Input Sources一欄下點加號添加新的輸入法：

Add an Input Source裏選擇Chinese，打開就可以看到Intelligent Pinyin與Pinyin。（如果打開沒有出現這些選項就重啓下電腦。） 然後你就可以愉快地輸入中文了。右擊語言欄也可以切換簡繁/全角半角什麼的。默認快捷鍵和Windows那邊一樣。
