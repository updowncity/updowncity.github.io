When working with complexes, many diagrams commute only up to homotopy. In order to have these diagrams actually commute, one need to work in a category where homotopy equivalences become isomorphisms. 想到我在前面寫过一則關於homotopy category的札記，然後因為MathJaX配置問題一氣之下把它給刪了，現在只好重寫（……） 先引用GTM281的記號：

2023-04-04

我怎么这么啰嗦。换新贴了（就是那个LaTeX吐槽贴）。

要么丢掉那些啰嗦的脚注，要么提前声明它们不计入总字数 「附录会有一个笔者的实验性构造。」救命啊别写这种只能扔进文学沙盒里的东西。 BGM都是素晴日chapter 2的，不知道为什么我对这种「不靠谱的救世主」很感兴趣，无论是初鸟创还是间宫卓司。（……话说我在现实中有个很诡异的特点就是…会一直在笑，就是一个自动的表情。i don’t know how to remove it…） 字数要求里的八千字如果指的是纯汉字的话就太恐怖了。（虽然我就是照着这个标准来的…以教务系统的死板程度……。）。

合理猜测我搞出这个东西是因为画不出来BBDG那本书里的图……我甚至觉得画TR4的图远比用它证东西需要更大的耐心。感谢维基百科提供了源代码，我可以把标准的先搬过去，然后用我那个容易画的。直接扔个别人没见过的真的很缺德。

時間戳部分丟失的不連續碎碎唸。依舊，複數的第一人稱代詞只是出於方便。至少在居民身份證與校園卡方面我們是同一的（它們都還在，沒搞丟～）

我依舊不知道數學系本科畢業論文有什麼約定俗成的規範是怎樣的。（至少）根據觀察與經驗，學術寫作不是文學，它是有一套關於措辭的語法規則的，儘管只會出現在寫作指南裡，（人們爲了維護體面不會把這種東西寫進成文的條例裡）它的地位與作用並不低於成文規定。（這也是作者用來隱藏起自己的面孔的工具。…………更何況我是數學系的，這種學科的論文裡出現強烈的個人風格會是一種很恐怖的事誒。）可是我到現在都還沒找到它。

每次想到同調代數，就很想來一句：「語言是用來表達事物之間的關係的，不是用來給人填堵的。」（前半句来自薇依）

t-structure的index真的坑死個人。（我懷疑…很少有人真的動手算過。） 就沒有人給那兩條Whitney條件修剪一下嗎…我這個簡化雖然沒什麼卵用但至少看上去順眼點（原始的那個…翻譯成中文以後更是又臭又長人鬼不識。） perverse t-structure那一節GTM281有個地方（p134）寫錯了…反覆確認，沒提conservative等價於non-degenerate這個結論就直接用（…而且它真的不是trivial的）這個結論挺重要的（至少能讓你擺脫那個反人類的index） 有的東西我覺得……具體寫出來會顯得很傻瓜（像在湊字數）…但略去不寫直接跳過去真的很缺德。（比如TR4的變形…雖然只是改變了字母的位置）

我真的很怕这个词。又一次(?)在不安定间删掉了一个post。感觉寫那些我就像一个小丑。真的好痛苦。

2023-03-23

闲逛的时候发现了AsciiDoc这个类似于Markdown优化的文本标记语言，然后想到org-mode：我可以把标题的syntax highlight给去掉，在第一级标题的地方写初始文本，然后从第二级标题开始放我那堆该死的括号1。很显然it doesn’t work因为我的括号是出现在段落内部的，而这些标题是必须位于段首的…寫出來會相當ugly。然后在想：我想要的是利用电脑屏幕(什么鬼)实现括号的折叠与配对（虽然这样一来出现于重叠的右括号的singularity就不复存在了，但至少：(1)阅读友好（幽灵能否读懂这坨狗屎我并不care但我寫論文還是需要把它們整理成條理清晰(至少能夠閱讀)的語言的……（我真的很想把充滿括號stack(s)的原始文本拿出去噁心別人就像現在這樣…）;(2)singularities并不是真的消失，它们只是以被消解(铺开)掉的形式呈现着（比如这种东西（但愿我没数错））））然后………………这TMD不就是lisp吗。（前兩天插上就被我忘掉的flag…） 其实我过去从来没想过会lisp，第一我对编写具体的程序不太感兴趣（我对事物的兴趣…很多时候是把原理搞清楚以后就结束了）；第二我在玩emacs的时候见到很多说lisp很难很不友好的(…信息?)；第三我真的找不到编程的动机。去年这个时候我因为选课失误而不得不学python然后怎么都学不会…很绝望，因为交不上作业。到结课的时候我还是不会寫python…（我很敬佩编程随想但我是真的很讨厌python因为学不会）其实我也无法理解，python作为语言本该是很trivial的东西（否则它不会成为被广泛使用的工具），我大一的时候也学过C和C++（是专必），也都还算顺利…。现在想想大概只是我无法适应封装(的世界)吧。刚才搜了下中文结果，那些知乎上对lisp吹捧或是贬低的文章让我感到……我不属于他们的世界。 另一个是，昨天我忽然意识到其实我至少从去年就已经确定了具体的方向，因为我无论做什么都会指向某个点……（我一直在寻找的）。发现以后觉得是显然的，但在此之前我一直很迷茫（以至于错过了一个申请季……我错过申请季不是因为脑子进水忘掉截止日期，而是因为对未来彻底的迷茫。）对我来说这些就好像是一个预先被安排好的人生轨迹，而非出于我个人的选择。 我只會寫這種心情随笔，小女生的碎碎唸。这里就像一个隐藏的秘密花园空间（我自知无力维护一个花园……）
